• Description

  • Well, here it is!

    Compactions CC for iOS11. This includes 84 icons. Note that not all CC modules are themed. The animated ones like the Lock, DND etc aren't. Not all CCModules options are themed as well (Mail didn't work for me, but I left it in case it works for someone else).

    I have only tested this on my iPhone X on iOS 11.3.1. I expect it'll work on Plus devices as well, but not the others.

    I'm honestly just swamped with university, so for the non-Plus/X devices, I apologize.

    !!!THIS PART IS IMPORTANT!!!

    The wifi and bluetooth icons are themed using BetterCCIcons. I've included both files inside the theme folder, so you guys need to move them (just the images, not the entire folder) to /Library/PreferenceBundles/BetterCCIcons.bundle and then enable glyph mode (experimental) in the settings

    As a last note, although this is free, donations are very welcome. I've spent a lot of effort and invested time into this for everyone. I hope it's to your liking!
